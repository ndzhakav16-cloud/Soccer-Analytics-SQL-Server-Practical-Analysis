# Soccer-Analytics-SQL-Server-Practical-Analysis
This project focuses on analyzing advanced soccer player data using Microsoft SQL Server.

# Project Overview
- The goal is to apply SQL fundamentals and intermediate techniques to extract insights on:
- Player performance
- Financial metrics
- Team-level statistics
- Demographics
- Player market valuatio

 # Aim of the Project
- To demonstrate the ability to analyze real-world sports data using SQL Server by writing accurate, optimized and professional SQL queries.

# Objectives

The tasks in this project aim to strengthen the following skills:
- Reading, exploring and understanding dataset structure
- Running summary statistics
- Aggregations (SUM, AVG, COUNT)
- Grouping & filtering (GROUP BY, ORDER BY, WHERE)
- Working with NVARCHAR numeric fields using TRY_CAST
- Creating calculated fields
- Ranking and sorting player performance
- Handling inconsistent data types
- Working with dates and contract years

# Technology Used

| Tool                                | Purpose                              |
| ----------------------------------- | ------------------------------------ |
| Microsoft SQL Server                | Query execution environment          |
| T-SQL                               | Querying, calculations, aggregations |
| SQL Server Management Studio (SSMS) | Running queries & exporting results  |
| CSV Dataset                         | Source data                          |
| GitHub                              | Project documentation & versioning   |
